# Práctica Final de Análisis y Modelado de Datos

### Máster en Data Science e IA — EVOLVE 2026

---

## 📋 Resumen

Esta práctica contiene **4 ejercicios** de análisis de datos y aprendizaje automático, enfocados en el análisis exploratorio de datos, regresión lineal y series temporales.

---

## 📚 Contenido

| Ejercicio | Título                        | Descripción                                                                         |
| --------- | ----------------------------- | ----------------------------------------------------------------------------------- |
| 1️⃣        | **Análisis Descriptivo**      | 📊 Analiza un dataset de precios de coches, genera gráficas y calcula correlaciones |
| 2️⃣        | **Regresión Lineal Simple**   | 🎯 Predice el precio usando el tamaño del motor con un solo predictor               |
| 3️⃣        | **Regresión Lineal Múltiple** | 🔧 Implementa regresión múltiple desde cero con `NumPy` (sin `scikit-learn`)            |
| 4️⃣        | **Series Temporales**         | ⏱️ Analiza y descompone una serie temporal sintética en componentes                 |

---

## 🚀 Cómo ejecutar

Cada ejercicio se ejecuta independientemente:

```bash
# Ejercicio 1
python ejercicio1_descriptivo.py

# Ejercicio 2
python ejercicio2_interferencia.py

# Ejercicio 3
python ejercicio3_regresion_multiple.py

# Ejercicio 4
python ejercicio4_series_temporales.py
```
