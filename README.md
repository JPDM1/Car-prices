# Práctica Final de Análisis y Modelado de Datos - Master en Data Science e IA EVOLVE 

El proyecto contiene 4 ejercicios de análisis de datos y aprendizaje automático.

## Contenido

**Ejercicio 1 - Análisis Descriptivo**

- Analiza un dataset de precios de coches
- Genera gráficas para cada atributo
- Calcula correlaciones y estadísticas

**Ejercicio 2 - Regresión Lineal Simple**

- Predice el precio de coches usando el tamaño del motor
- Usa un solo predictor: Enginesize
- Calcula métricas de evaluación (MAE, RMSE, R²)

**Ejercicio 3 - Regresión Lineal Múltiple**

- Implementa regresión múltiple desde cero con NumPy
- Usa la fórmula matemática de mínimos cuadrados
- No usa librerías de machine learning

**Ejercicio 4 - Series Temporales**

- Analiza una serie temporal sintética
- Descompone la serie en tendencia, estacionalidad y ruido
- Verifica si el ruido se comporta como un ruido ideal

## Cómo ejecutar

Cada ejercicio se ejecuta con:

```bash
python ejercicio1_descriptivo.py
python ejercicio2_interferencia.py
python ejercicio3_regresion_multiple.py
python ejercicio4_series_temporales.py
```

## Archivos

- `data/`: Contiene el dataset de coches
- `output/`: Guarda los gráficos y resultados
- `Respuestas.md`: Respuestas detalladas de cada ejercicio
- `anejos.py`: Funciones auxiliares para colores en la terminal
